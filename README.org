#+title: Kugire Annotations for the Kokinwakashu

* Overview

A kugire (句切れ) annotation layer for the Kokinwakashu (古今和歌集) TEI/XML data.
Kugire marks a phrase or clause break within a poem — a judgement that is often
ambiguous. Following Asaoka, two types are distinguished:

1. *Grammatical break* — derivable from morphological analysis of the poem text.
2. *Semantic break* — derivable from explanatory translations by waka experts.

Because different translators may disagree, the annotation schema allows multiple
competing kugire annotations on the same poem, each tagged with its evidence source.

* Tool design

The pipeline is organised as pure transformation functions composed by a thin I/O
shell. I/O (file reads, editor, XML writes) is isolated at the edges.

** Data type

#+begin_src go
type PoemData struct {
    ID           int
    XML          string            // raw TEI <l> fragment
    Morphemes    []Token           // <w> elements from kokin-annotated.xml
    Translations map[string]string // translator_code -> translation text
}

type KugirePos struct {
    LineIdx int    // 0-based index into the 5 poem lines
    Source  string // "morph" | translator code
}
#+end_src

** Composed review workflow

#+begin_src go
func ReviewPoem(id int, translatorCode string) error {
    data         := LoadPoem(id)
    suggestions  := append(SuggestMorph(data), SuggestSeman(data, translatorCode)...)
    path         := WriteDraft(RenderDraft(data, suggestions))
    OpenEditor(path)
    positions    := ParseDraft(ReadDraft(path))
    if len(DiffKugire(suggestions, positions)) == 0 { return nil }
    return SavePoem(id, AnnotateXML(data.XML, positions))
}
#+end_src

* Draft format

The nano draft is a plain 5-line poem. A kugire is marked with an end-of-line
tag =​[K:<source>]=. The user adds or removes tags to adjust the suggestion.

#+begin_example
春はあけぼの [K:morph]
やうやうしろく
なりゆく山ぎは [K:seman]
小紫野も
すこしあかりて
#+end_example

Source codes:
- =morph= — grammatical break from morphological analysis
- translator code (e.g. =kaneko=) — semantic break from that expert's translation
